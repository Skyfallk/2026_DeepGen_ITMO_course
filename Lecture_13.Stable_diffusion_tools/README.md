# Инструменты для генерации

## Лидерборды генеративных моделей

- [ArtificialAnalysis Arena (Text-to-Image) Leaderboard](https://artificialanalysis.ai/text-to-image/arena?tab=leaderboard) — сравнение моделей генерации изображений, там же есть для видео и речи.
- [IMGSYS Rankings](https://imgsys.org/rankings) — другой рейтинг моделей с разными дообученными чекпоинтами SD.
- [Civitai](https://civitai.com/) - дообученные чекпоинты, workflow и др. можно брать тут.

## Модели генерации изображений

- **Пример реалистичной SDXL модели**: [Civitai](https://civitai.com/images/5374513)
- **Лучшая open-source модель сейчас**: [FLUX.1-dev от Black Forest Labs](https://huggingface.co/black-forest-labs/FLUX.1-dev)

## Интерфейсы (UI) для генерации изображений

- [AUTOMATIC1111 Web UI](https://github.com/AUTOMATIC1111/stable-diffusion-webui?tab=readme-ov-file) — популярный веб-интерфейс для работы с Stable Diffusion.
- [ComfyUI](https://github.com/comfyanonymous/ComfyUI?tab=readme-ov-file) — модульный визуальный интерфейс для создания графов генерации.
- [Phygital+](https://app.phygital.plus/) — облачный интерфейс с визуальным графовым подходом, аналог ComfyUI.

## Хостинг ComfyUI, если нет своей видеокарты (нужна иностранная карта, но русские сервисы дороже)

- [RunPod](https://www.runpod.io/) — облачный сервис с доступом к GPU по требованию, поддерживает различные шаблоны, включая ComfyUI.
- [RunComfy](https://runcomfy.com/) — специализированный сервис для запуска ComfyUI в облаке с предустановленными узлами и моделями.

## Апскейлеры (повышение качества изображений)

- [OpenModelDB](https://openmodeldb.info/) — база данных моделей для апскейлинга изображений.
- [Topaz Gigapixel AI](https://www.topazlabs.com/gigapixel) — коммерческий инструмент для увеличения разрешения изображений с использованием ИИ.

## Некоторые модели для генерации видео

- [WANXiang Video Creation](https://wan.video/wanxiang/videoCreation)
- [OpenAI Sora](https://help.openai.com/en/articles/9957612-generating-videos-on-sora)
